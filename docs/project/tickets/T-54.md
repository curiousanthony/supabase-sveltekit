Restructure the Fiche tab to expose the mandatory pédagogique and tarification fields (currently invisible despite existing in the schema), hide the Référencement section behind a teaching toggle (Hick's Law — most formations aren't certifying), and remove the Financement card entirely (moved to Finances in T-53).

New section order:

1. **Identification** (intitulé, type, modalité, durée, dates) — already exists, kept as-is
2. **Référencement** (collapsed, conditional on toggle "Cette formation prépare-t-elle à une certification ?")
3. **Pédagogique** — new card
4. **Logistique** (lieu, accessibilité — accessibilité sub-card lands in T-56)
5. **Client / Commercial** — moves `prixPublic` + `prixConvenu` here

## acceptance

- Entire "Financement" card removed from Fiche
- "TJM formateur" input removed (column dropped in T-51)
- New "Pédagogique" Card with: read-only `ObjectifsRollup` aggregating `modules[].objectifs` from Programme + editable textareas for `prerequis` and `publicVise` (no formation-level `objectifs` editor — that data lives on modules; no `modalitesEvaluation` Select — field does not exist on `formations` table)
- The two editable textareas use **explicit Modifier/Enregistrer buttons** (NOT inline blur-save) per UX review §"Editing pattern" rationale (Marie types long text, gets interrupted)
- `ObjectifsRollup` shows count badge + first 3 module objectifs + "Modifier dans Programme →" CTA; empty state when no modules
- New "Tarification" Card with `prixPublic` + `prixConvenu` numeric inputs (inline-edit pattern OK for short values)
- "Référencement" Card hidden behind a Switch/toggle "Cette formation prépare-t-elle à une certification ?" — when toggled on, shows `codeRncp` + `codeRs` + `codeCpfFiche` + `niveauQualification` + `certificateur` + `dateEnregistrementRncp`, each with a `Popover` info icon (popover copy ships in T-58)
- Section order matches the design above
- `allowedFields` in `+page.server.ts` updated: financing fields removed, pédagogique + tarification + référencement fields added
- Inline-cell idle hover affordance polished: dotted underline + Pencil icon on hover
- All copy in French ; Marie persona vocabulary
