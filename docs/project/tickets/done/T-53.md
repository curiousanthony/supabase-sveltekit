Replace the current "Revenus" 3-tile card on the Finances tab with the new financial cockpit: a synthesis card surfacing the funding posture at a glance, a "Sources de financement" card listing every funding line with its lifecycle, and a dialog for add/edit. Invoices gain a "Lier à une source de financement" picker.

The synthesis card includes a **per-payer split** for "Reste à charge" so Marie sees exactly who owes what (apprenant vs entreprise vs OF) without leaving the page.

## acceptance

- New synthesis card with 4 tiles: Total formation (= `prixConvenu`) | Financé (sum granted) | Reste à charge (with per-payer mini-breakdown: chip per payer with non-zero balance) | Statut global Badge (Entièrement financé / Partiellement financé / Sans financement / En attente)
- "Sources de financement" Card lists all funding lines: source label (translated from enum), payer_label + payer_type chip, montants requis/accordé, status Badge, dossier ref, edit/delete icon buttons
- "+ Ajouter une source de financement" Dialog with all fields: source (`Combobox` with canonical list grouped by category), payer_label (text), payer_type (`Select`: apprenant/entreprise/OF), requestedAmount, grantedAmount, status (`Select`), decisionDate, expectedPaymentDate, dossierReference (text), notes (textarea)
- Smart defaults in the dialog: source=CPF → payer_type=apprenant, payer_label="Caisse des Dépôts" ; source=OPCO_* → payer_type=entreprise, payer_label prefilled with workspace OPCO if known ; source=EmployeurDirect → payer_type=entreprise, payer_label=client company name
- Empty state with explicit CTA when no funding sources
- Invoice dialog gains a `Combobox` "Lier à une source de financement" (optional, lists only same-formation sources)
- Old "Revenus" 3-tile card removed
- Loss-aversion phrasing for unfunded balance ("Reste à charge à recouvrer", never "Manque à gagner"); colors per UX review §"Synthesis card visual language"
- Mobile (≤640px): synthesis tiles stack 2×2 ; per-payer chips wrap below the tile ; funding rows collapse to two lines (source + payer on line 1, amounts + status on line 2)
