Existing users who land on Finances after the redesign should immediately understand that the financing UI moved out of the Fiche. A one-time card banner explains the change with a friendly CTA, then disappears for good (per user, per workspace).

## acceptance

- Card banner renders at the top of Finances tab when `localStorage.getItem('finances-redesign-banner-v1-dismissed')` is null
- Dismiss button persists to localStorage AND writes one audit_log entry per workspace user (so we can measure rollout)
- Banner copy verbatim per UX review §"Migration moment"
- Banner respects dark mode and reduced-motion preferences
- Doesn't render at all on first-ever visit (no formations yet) to avoid noise on empty workspaces
