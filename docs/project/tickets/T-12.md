Per-document stale indicators (not global banner). When `formation.updatedAt > document.generatedAt`, show inline warning: "Les données ont changé — [Voir les changements] [Régénérer]". Versioning rules: `genere` (never sent) → replace in place; `envoye` or later → new row, old gets `remplace`; `signe` → block regeneration (suggest avenant).

## acceptance

- Per-document stale indicators appear when formation data changed since generation
- Inline warning with "Voir les changements" and "Régénérer" actions
- Replace in place when status is `genere`
- Create new row and mark old as `remplace` when status is `envoye` or later
- Block regeneration for `signe` documents with guidance toward avenant
