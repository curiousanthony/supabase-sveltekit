Marie should never feel stupid asking "what is this code?". Each référencement field on the Fiche gets a Popover with a one-sentence French explanation + a link to the official registry. Copy is verbatim per the UX review §"Référencement micro-copy".

## acceptance

- `Popover` info icon (Lucide `Info` or `HelpCircle`) next to each label: Code RNCP, Code RS, Code CPF, Certificateur, Niveau de qualification
- Copy verbatim per UX review §"Référencement micro-copy" — concise, human, French, never patronising
- External links open in new tab with `rel="noopener noreferrer"` (France Compétences, Mon Compte Formation registry)
- Popovers keyboard-accessible (`aria-describedby`, focus trap respects Popover patterns from shadcn)
- Mobile: popovers anchored sensibly to avoid edge clipping
