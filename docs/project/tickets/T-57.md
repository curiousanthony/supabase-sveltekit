Once the Financement section leaves the Fiche (T-53/T-54), Marie still needs to know the funding posture from any sub-route without navigating. A read-only chip in the formation header (above the tab strip) carries the synthesis: source(s), accordée count, reste à charge total. Click → deep link to `/formations/{id}/finances`.

## acceptance

- Chip placed in the formation header layout (`src/routes/(app)/formations/[id]/+layout.svelte`), visible on every sub-route
- Uses `getFundingSummary(formationId)` from `src/lib/services/funding-summary.ts` (built in T-52)
- Format example: `OPCO Atlas — 2/3 sources accordées · Reste à charge : 600 €`
- Color rules: green (fully funded) / amber (partial or in flight) / grey (no funding)
- Click navigates to `/formations/{id}/finances` (full-area click, not just an icon)
- Accessible: descriptive `aria-label` summarising the same info, focusable via keyboard
- Loads via `+layout.server.ts` so chip is SSR-rendered (no hydration flash)
