Once the Financement section leaves the Fiche (T-53/T-54), Marie still needs to know the funding posture without navigating. The April-21 UX review proposed a chip in the SiteHeader; stakeholder review on 2026-04-22 rejected that placement (header is `shrink-0` at fixed `--header-height` with no slack, and the proposed colors collided with the existing `formation.statut` Badge + colored HudBanner). Phase 1 was re-run with product-analyst + ux-designer; the user picked the **D + F composite**: a colored dot on the Finances tab (visible from every sub-route via the existing tab strip) + an upgraded "Financement" card on the Aperçu page (Marie's morning landing) showing the canonical multi-source summary. The signal is operational — not Qualiopi-mandatory (analyst §3) — so a glanceable indicator on the always-sticky tab strip with full detail one click away satisfies the user need without touching header chrome.

## acceptance

- The Finances tab in the formation `+layout.svelte` shows a color-coded dot (emerald = `Entièrement financé`, amber = `Partiellement financé` or `En attente`, none = `Sans financement` with no pricing context). The existing red `overdueInvoices` dot wins when both apply.
- Hover or focus on the Finances tab opens a shadcn `Tooltip` showing the statut label + reste à charge or pending amount in € (`Intl.NumberFormat('fr-FR')` formatted).
- Per-tab `aria-label` is built server-side and includes the funding statut + euro amount in screen-reader-friendly form (e.g. *"Finances. Partiellement financé, 600 euros restent à charge."*); when overdue, announces "Facture en retard." with no funding clause; default "Finances" on `Sans financement`.
- The Aperçu page "Résumé financier" card (renamed "Financement") replaces its body with the canonical 3-tile grid (Total · Financé · Reste à charge) + statut pill + top-3 sources list + "+N autres" link + Marge prévisionnelle line. Pulled from `data.fundingSummary` provided by the layout loader.
- Two distinct empty states on the Aperçu card: (a) `prixConvenu` set + zero sources → "Prix convenu : X € — aucun financeur saisi." with "Ajouter une source de financement" link; (b) both null → muted "Aucune information de tarification renseignée." copy with no pill, no color (avoids "free internal training vs unfunded" confusion).
- All values are SSR-rendered via `+layout.server.ts` extending the existing formation query with `fundingSources: true` (zero new round-trips) and calling `getFundingSummary(fundingSources, prixConvenu)` from `src/lib/services/funding-summary.ts`.
- The SiteHeader is **not** modified (no chip, no new pill); the HudBanner is **not** modified (its quest-only contract is preserved); no new sticky band is introduced. Header height is unchanged on every viewport.
- Mobile (375 px): dot stays visible at 6 px; tooltip degrades to long-press; Aperçu tiles stack to `grid-cols-1`.
