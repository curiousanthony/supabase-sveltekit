Add the missing référencement fields to the schema so the Fiche Référencement card (T-54) can render them. Code RNCP alone covers only a minority of certifying OFs — Code RS (Répertoire Spécifique: TOSA, PIX, SST, PCIE...) is mandatory for many common certifications, and Code CPF (numéro de fiche Mon Compte Formation) is required for any CPF-eligible formation.

## acceptance

- Schema additions to `formations` table: `code_rs` text, `code_cpf_fiche` text, `niveau_qualification` integer (CHECK between 1 and 8), `certificateur` text, `date_enregistrement_rncp` date
- All five fields exposed via the Fiche Référencement card (built in T-54)
- `allowedFields` in `src/routes/(app)/formations/[id]/fiche/+page.server.ts` extended to include the five new fields
- Placeholder copy hints at the expected format (e.g. `RS5832`, `285568`)
- Migration includes a down step that drops the columns
