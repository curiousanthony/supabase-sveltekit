# Formations — Pré-vol avant génération de documents

> **Pour qui ?** Secrétaires, responsables pédagogiques (persona Marie)
>
> **En bref :** Avant de générer un PDF (devis, convention, convocation, feuille d’émargement, certificat…), Mentore Manager affiche une **liste de contrôle** : ce qui est prêt (vert), ce qui manque avec un lien **« Compléter → »** vers le bon onglet, et ce qui est **bloquant** vs **simple avertissement**.

---

## Contrôle avant génération

Lorsque vous lancez une génération depuis l’onglet **Documents** ou depuis le **Suivi** d’une formation :

1. Une boîte de dialogue résume l’état des prérequis (client / entreprise, séance pour l’émargement, éléments optionnels comme le logo, etc.).
2. Les **blocages** empêchent la génération tant que les informations juridiquement nécessaires ne sont pas renseignées.
3. Les **avertissements** signalent des données utiles mais non obligatoires ; vous pouvez quand même générer, en acceptant explicitement l’avertissement si l’interface le demande.

Les liens **« Compléter → »** ouvrent la fiche formation au bon endroit pour corriger rapidement l’erreur.

---

## Reprendre après correction

Si vous étiez en train de générer un document et que vous avez dû compléter des champs, un **bandeau discret** peut vous proposer de **reprendre** la génération une fois les données corrigées, sans repartir de zéro.

---

## Conformité Qualiopi (lecture rapide)

Cette étape réduit les oublis de pièces justificatives et les PDF générés sur des données incomplètes — utile pour les indicateurs portant sur la **traçabilité** des parcours et des preuves administratives.
