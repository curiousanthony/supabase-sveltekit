## 🎯 Pourquoi cette fonctionnalité ?

Vous ne savez plus quel OPCO gère quelle entreprise ? Vous cherchez le SIRET d'un client pendant 10 minutes ? Vous oubliez qu'une entreprise a déjà acheté 5 formations l'année dernière ?

**Le problème avec les autres outils :**
- **Digiforma** : Pas de gestion d'entreprises, tout est dans les formations de manière éparpillée
- **Qualiobee** : Fiche entreprise basique, impossible de voir l'historique complet
- **Dendreo** : Tellement de champs à remplir que personne ne le fait

**La solution Mentore Manager :**

Une fiche entreprise complète mais simple : nom, SIRET, secteur, taille, OPCO, budget... et surtout : tous les contacts, tous les Deals, toutes les formations au même endroit. Vous voyez en 1 coup d'œil tout l'historique d'un client.

**Bénéfices clés :**
- ✅ **Fiche complète** : Toutes les infos juridiques, commerciales et financières
- ✅ **SIRET et OPCO** : Informations essentielles pour le financement et la facturation
- ✅ **Historique complet** : Tous les Deals et formations de cette entreprise
- ✅ **Multi-contacts** : Voir tous les interlocuteurs d'une même entreprise
- ✅ **Budget et financement** : Budget estimé, dispositifs (OPCO, CPF, FNE...)

## 📋 Fonctionnalités principales

### Créer une entreprise

**C'est quoi ?**
C'est enregistrer les informations d'une société, organisme ou association cliente.

**Pourquoi vous en avez besoin ?**
Pour centraliser toutes les informations d'un client et voir son historique commercial.

**Comment l'utiliser ?**
1. Ouvrez CRM → Entreprises
2. Cliquez sur "Nouvelle entreprise"
3. Remplissez les informations essentielles :
   - **Nom** : Le nom commercial (ex: TechCorp SAS)
   - **SIRET** : 14 chiffres (obligatoire pour facturation)
   - **Statut juridique** : SAS, SARL, SA, Association...
   - **Taille** : Nombre de salariés (1-10, 11-50, 51-250...)
   - **Secteur d'activité** : Informatique, Santé, Commerce...
4. Ajoutez les informations complémentaires (optionnel) :
   - **Adresse complète** : Rue, code postal, ville, région
   - **Site web** : L'URL du site
   - **Budget estimé** : Pour qualifier le potentiel
   - **OPCO** : L'organisme de financement (Akto, Atlas, OPCO EP...)
   - **Dispositifs** : CPF, FNE, Plan de développement des compétences...
   - **Notes internes** : Informations confidentielles
5. Cliquez sur "Enregistrer"

**Astuces :**
- 💡 **SIRET essentiel** : Obligatoire pour les formations financées et la facturation
- 💡 **OPCO stratégique** : Ça accélère les ventes si vous savez qui finance
- 💡 **Budget estimé** : Pour prioriser vos efforts commerciaux (gros vs petits clients)
- 💡 **Notes confidentielles** : "Négocient toujours -10%", "Payent en 60 jours"...

---

### Lier des contacts à une entreprise

**C'est quoi ?**
C'est indiquer quelles personnes travaillent pour cette entreprise.

**Pourquoi vous en avez besoin ?**
Pour savoir qui contacter dans l'entreprise et avoir tous les interlocuteurs sous la main.

**Comment l'utiliser ?**
1. Ouvrez la fiche entreprise
2. Section "Contacts liés"
3. Cliquez sur "Lier un contact"
4. Sélectionnez le contact dans la liste (ou créez-le)
5. Le contact apparaît dans la liste

**Astuces :**
- 💡 **Plusieurs contacts** : Une grande entreprise peut avoir 5-10 contacts (DRH, Directeur, Responsables...)
- 💡 **Rôles clairs** : Mettez le poste dans la fiche contact pour savoir qui fait quoi
- 💡 **Contact principal** : Identifiez le décideur avec une note "Contact principal"

---

### Modifier une entreprise

**C'est quoi ?**
C'est mettre à jour les informations de l'entreprise (changement d'adresse, nouveau SIRET, changement d'OPCO...).

**Pourquoi vous en avez besoin ?**
Les entreprises évoluent : fusion, déménagement, changement de statut...

**Comment l'utiliser ?**
1. Ouvrez la fiche entreprise
2. Cliquez sur le champ à modifier (édition inline)
3. Modifiez
4. Cliquez ailleurs pour enregistrer automatiquement

**Astuces :**
- 💡 **SIRET changé ?** : En cas de fusion/acquisition, mettez le nouveau SIRET
- 💡 **Notes historiques** : "Fusion avec XYZ en mars 2026" pour traçabilité
- 💡 **OPCO change** : L'OPCO dépend du secteur, si l'activité change, l'OPCO aussi

---

### Voir l'historique d'une entreprise

**C'est quoi ?**
C'est voir tous les Deals et formations liés à cette entreprise.

**Pourquoi vous en avez besoin ?**
Pour préparer une relance commerciale, connaître l'historique client, identifier des opportunités de montée en gamme.

**Comment l'utiliser ?**
1. Ouvrez la fiche entreprise
2. Faites défiler vers le bas
3. Section "Deals liés" : Toutes les opportunités commerciales
4. Section "Formations réalisées" : Toutes les formations passées (via les Deals gagnés)
5. Analysez :
   - Fréquence d'achat : 1 fois/an, 3 fois/an ?
   - Montant moyen : Quel budget ?
   - Thématiques : Quels sujets ? Bureautique, Management, Langues ?

**Astuces :**
- 💡 **Pattern d'achat** : "Achète toujours en septembre pour janvier" → relancez en août
- 💡 **Montée en gamme** : "A fait Niveau 1, proposez Niveau 2"
- 💡 **Cross-sell** : "A fait Excel, proposez PowerPoint"

---

### Rechercher et filtrer les entreprises

**C'est quoi ?**
C'est trouver rapidement une entreprise par son nom, SIRET, secteur ou taille.

**Pourquoi vous en avez besoin ?**
Pour cibler vos actions commerciales : "Toutes les entreprises informatiques de 50-250 salariés" = prospection ciblée.

**Comment l'utiliser ?**
1. Dans la liste des entreprises, utilisez la barre de recherche
2. Tapez : nom, ville, SIRET
3. Ou utilisez les filtres :
   - **Par secteur** : Toutes les entreprises IT
   - **Par taille** : 50-250 salariés
   - **Par OPCO** : Toutes les entreprises avec OPCO Akto
4. Les résultats s'affichent instantanément

**Astuces :**
- 💡 **Prospection ciblée** : Secteur Santé + Taille 100-250 = cible parfaite pour une offre
- 💡 **OPCO groupé** : Toutes les entreprises OPCO EP = 1 seul dossier de financement simplifié
- 💡 **Géographie** : Filtrer par région/ville pour optimiser les déplacements

---

### Supprimer une entreprise

**C'est quoi ?**
C'est effacer définitivement une entreprise de votre base.

**Pourquoi vous en avez besoin ?**
Pour supprimer les doublons ou les entreprises qui ont fermé.

**Comment l'utiliser ?**
1. Ouvrez la fiche entreprise
2. Menu actions (3 points) → "Supprimer"
3. Confirmez

**Astuces :**
- 💡 **Vérifiez les liens** : Si l'entreprise a des Deals ou formations, ils seront déliés
- 💡 **Archivez plutôt** : Ajoutez une note "Entreprise fermée en 2026" pour l'historique
- 💡 **Doublons** : Si 2 fiches pour la même entreprise, supprimez la moins complète

## 👤 Cas d'usage

### Scénario 1 : Nouveau client prospect

**Situation :**
Sophie reçoit une demande de TechCorp, une entreprise informatique de 120 salariés à Lyon.

**Étapes :**
1. Sophie ouvre CRM → Entreprises
2. Elle clique sur "Nouvelle entreprise"
3. Elle recherche le SIRET de TechCorp sur societe.com
4. Elle remplit :
   - Nom : TechCorp SAS
   - SIRET : 12345678901234
   - Statut : SAS
   - Secteur : Informatique
   - Taille : 51-250 salariés
   - Adresse : 123 rue de Lyon, 69001 Lyon, Auvergne-Rhône-Alpes
   - Site web : https://techcorp.fr
   - OPCO : Akto (OPCO IT)
   - Budget estimé : 50 000€/an
5. Elle enregistre
6. Elle lie le contact Marc Dupont (DRH) qu'elle a créé avant
7. Elle crée un Deal : "Formation Excel TechCorp"

**Résultat :**
En 5 minutes, Sophie a une fiche complète avec toutes les infos pour vendre et facturer. Tout est centralisé.

---

### Scénario 2 : Préparer une relance client

**Situation :**
Thomas veut relancer FormaPro qui n'a pas acheté de formation depuis 1 an.

**Étapes :**
1. Thomas ouvre la fiche FormaPro
2. Il regarde l'historique :
   - 3 formations en 2025 : Excel, PowerPoint, Word
   - Montant total : 4500€
   - Dernier achat : février 2025
3. Il identifie une opportunité : "Ils ont fait Pack Office, et si on proposait Excel Avancé ?"
4. Il prend les contacts liés : Julie Martin (Responsable formation)
5. Il l'appelle : "Bonjour Julie, l'année dernière vous aviez fait notre Pack Office. J'ai une offre Excel Avancé qui pourrait vous intéresser..."
6. Julie est intéressée !
7. Thomas crée un Deal : "Excel Avancé FormaPro"

**Résultat :**
Grâce à l'historique complet, Thomas a personnalisé son approche commerciale. Deal créé, Formation vendue.

---

### Scénario 3 : Gérer le financement OPCO

**Situation :**
Marie doit créer une formation pour 3 entreprises différentes, toutes avec OPCO Akto.

**Étapes :**
1. Marie filtre les entreprises par "OPCO : Akto"
2. Elle obtient 12 entreprises
3. Elle identifie ses 3 clients : TechCorp, DigitalPro, WebAgency
4. Elle crée les 3 formations
5. Pour chacune, l'OPCO est automatiquement rempli (récupéré de la fiche entreprise)
6. Elle prépare UN SEUL dossier OPCO Akto pour les 3 formations

**Résultat :**
Marie a gagné 2 heures en évitant de chercher l'OPCO de chaque client et en regroupant les dossiers.

## ❓ Questions fréquentes

**Q: Le SIRET est-il obligatoire ?**  
R: Oui pour les entreprises françaises (nécessaire pour facturation et financement). Pour les clients étrangers, vous pouvez le laisser vide.

**Q: Comment trouver le SIRET d'une entreprise ?**  
R: Sur societe.com, infogreffe.fr ou le site de l'INSEE. C'est public et gratuit.

**Q: Quelle est la différence entre une Entreprise et un Contact ?**  
R: **Entreprise** = personne morale (TechCorp SAS). **Contact** = personne physique (Marc Dupont qui travaille chez TechCorp).

**Q: Puis-je créer une entreprise sans contact ?**  
R: Oui, mais c'est mieux d'avoir au moins 1 contact pour pouvoir communiquer.

**Q: Comment gérer les changements de SIRET (fusion, acquisition) ?**  
R: Créez une nouvelle fiche entreprise avec le nouveau SIRET. Gardez l'ancienne avec une note "Fusionné avec XYZ" pour l'historique.

**Q: L'OPCO, c'est obligatoire ?**  
R: Non, mais c'est très utile pour le financement. Si vous ne savez pas, laissez vide.

**Q: Comment trouver l'OPCO d'une entreprise ?**  
R: Ça dépend du secteur d'activité. Mentore Manager suggère l'OPCO en fonction du secteur que vous choisissez.

**Q: Combien d'entreprises puis-je créer ?**  
R: Illimité ! Gérez autant d'entreprises que vous voulez.
