# Questionnaires

> **Pour qui ?** Directeurs, Secrétaires, Formateurs
> 
> **En bref :** Créez et gérez vos tests de positionnement, quiz et audits des besoins avec lien vers vos outils externes.
> 
> **Indicateurs Qualiopi :** Indicateur 3 (analyse des besoins), Indicateur 6 (modalités d'évaluation)

---

## 🎯 Pourquoi cette fonctionnalité ?

Vous créez vos quiz dans Google Forms mais vous ne savez plus où sont les liens ? Vous cherchez pendant 10 minutes quel questionnaire utiliser pour quelle formation ? Vous oubliez d'envoyer les tests de niveau aux apprenants ?

**Le problème avec les autres outils :**
- **Digiforma** : Pas de gestion de questionnaires externes, tout doit être dans leur système propriétaire
- **Qualiobee** : Interface confuse, impossible de savoir quel questionnaire est lié à quelle formation
- **Dendreo** : Pas de lien avec Google Forms ou Tally, vous devez tout recréer chez eux

**La solution Mentore Manager :**

Vous créez vos quiz là où vous voulez (Google Forms, Tally, Typeform...). Mentore Manager centralise juste les liens et les relie à vos programmes et modules. Besoin du test de niveau Excel ? Cliquez, le lien s'ouvre. Simple, efficace.

**Bénéfices clés :**
- ✅ **Tous vos quiz au même endroit** : Fini de chercher dans vos emails et favoris
- ✅ **Lien automatique** : Attachez un questionnaire à un programme, il apparaît dans toutes les formations
- ✅ **Outil de votre choix** : Google Forms, Tally, Typeform... utilisez ce que vous aimez
- ✅ **3 types clairs** : Test de niveau, Quiz, Audit des besoins - pas de confusion
- ✅ **Conformité Qualiopi** : Preuve que vous évaluez les besoins et les compétences

## 📋 Fonctionnalités principales

### Créer un questionnaire

**C'est quoi ?**
C'est enregistrer un lien vers votre quiz (Google Forms, Tally...) et le catégoriser pour le retrouver facilement.

**Pourquoi vous en avez besoin ?**
Pour centraliser tous vos questionnaires et les lier à vos programmes et modules. Plus besoin de chercher "le lien du test Excel" dans vos emails.

**Comment l'utiliser ?**
1. Créez d'abord votre quiz sur Google Forms ou Tally (Mentore Manager ne crée pas le quiz, juste le lien)
2. Copiez l'URL de votre quiz
3. Dans Mentore Manager, allez dans Bibliothèque → Onglet "Questionnaires"
4. Cliquez sur "Nouveau questionnaire"
5. Remplissez :
   - **Titre** : Le nom de votre quiz (ex: "Test de niveau Excel")
   - **Type** : Cliquez sur la carte correspondante :
     - **Test de niveau** : Pour évaluer le niveau initial (positionnement)
     - **Quiz / Exercice** : Pour évaluer pendant ou après la formation
     - **Audit des besoins** : Pour identifier les besoins avant la formation
   - **URL du test** : Collez l'URL de votre Google Forms ou Tally
6. Liez à vos programmes et/ou modules (optionnel) :
   - Cliquez sur les programmes concernés
   - Cliquez sur les modules concernés
7. Cliquez sur "Créer le questionnaire"

**Astuces :**
- 💡 **Créez le quiz d'abord** : Mentore Manager stocke juste le lien, créez le contenu sur Google Forms
- 💡 **URL sécurisée** : Utilisez des liens https://, pas http://
- 💡 **Nommage clair** : "Test Excel Niveau 1" plutôt que "Test 1" - vous en aurez plusieurs
- 💡 **Liez immédiatement** : Attachez tout de suite à vos programmes/modules, vous gagnerez du temps

---

### Utiliser un questionnaire

**C'est quoi ?**
C'est ouvrir le lien du questionnaire pour l'envoyer aux apprenants ou le remplir vous-même.

**Pourquoi vous en avez besoin ?**
Pour accéder rapidement au quiz sans chercher dans vos favoris ou emails.

**Comment l'utiliser ?**
1. Dans la liste des questionnaires, trouvez celui que vous voulez
2. Cliquez sur le lien (icône lien externe) dans la colonne "Lien"
3. Le quiz s'ouvre dans un nouvel onglet
4. Copiez l'URL pour l'envoyer par email, ou envoyez directement le lien

**Astuces :**
- 💡 **Lien dans les convocations** : Copiez le lien pour l'inclure dans vos emails de convocation
- 💡 **Menu déroulant** : Vous pouvez aussi cliquer sur les 3 points → "Ouvrir le lien"
- 💡 **Statistiques externes** : Les résultats sont dans Google Forms, pas dans Mentore Manager

---

### Modifier un questionnaire

**C'est quoi ?**
C'est changer le nom, le type ou le lien d'un questionnaire existant.

**Pourquoi vous en avez besoin ?**
Vous avez recréé votre quiz sur un nouvel outil, vous voulez changer le lien. Ou vous avez mal catégorisé le type.

**Comment l'utiliser ?**
1. Cliquez sur le nom du questionnaire
2. Modifiez ce que vous voulez
3. Cliquez sur "Enregistrer"

**Astuces :**
- 💡 **URL cassée ?** : Si votre Google Forms a expiré, mettez le nouveau lien
- 💡 **Liez/déliez** : Ajoutez ou retirez des programmes et modules librement
- 💡 **Changez le type** : Si vous vous êtes trompé de catégorie, corrigez

---

### Supprimer un questionnaire

**C'est quoi ?**
C'est effacer un questionnaire que vous n'utilisez plus.

**Pourquoi vous en avez besoin ?**
Pour garder votre liste propre et ne pas vous perdre dans des centaines de vieux questionnaires.

**Comment l'utiliser ?**
1. Ouvrez le questionnaire
2. En bas de page, section "Supprimer ce questionnaire"
3. Cliquez sur "Supprimer"
4. Confirmez

**Astuces :**
- 💡 **Attention** : C'est irréversible mais ça ne supprime PAS votre Google Forms, juste le lien dans Mentore Manager
- 💡 **Vérifiez l'usage** : Regardez à quels programmes/modules il est lié avant de supprimer

## 👤 Cas d'usage

### Scénario 1 : Créer un test de niveau Excel

**Situation :**
Marie donne des formations Excel. Elle veut envoyer un test de niveau AVANT la formation pour adapter le contenu au groupe.

**Étapes :**
1. Marie crée son quiz sur Google Forms avec 15 questions Excel (elle prend 20 minutes)
2. Elle copie l'URL du Google Forms
3. Dans Mentore Manager, Bibliothèque → Questionnaires
4. Elle clique sur "Nouveau questionnaire"
5. Elle remplit :
   - Titre : "Test de niveau Excel"
   - Type : "Test de niveau"
   - URL : elle colle son lien Google Forms
6. Elle clique sur son programme "Pack Office Complet" pour le lier
7. Elle clique sur ses modules "Excel Niveau 1" et "Excel Niveau 2"
8. Elle enregistre

**Résultat :**
Maintenant, chaque fois que Marie crée une formation "Pack Office", le lien vers le test de niveau apparaît automatiquement. Elle peut l'envoyer aux apprenants en 2 clics.

---

### Scénario 2 : Faire un audit des besoins

**Situation :**
Une entreprise veut une formation sur-mesure. Sophie doit identifier leurs besoins précis avant de construire le programme.

**Étapes :**
1. Sophie a déjà créé un questionnaire "Audit besoins bureautique" sur Tally il y a 2 mois
2. Elle ouvre Bibliothèque → Questionnaires
3. Elle trouve "Audit besoins bureautique"
4. Elle clique sur le lien externe
5. Le Tally s'ouvre, elle copie l'URL
6. Elle l'envoie par email à son contact dans l'entreprise
7. L'entreprise remplit le questionnaire
8. Sophie analyse les réponses sur Tally
9. Elle crée un programme sur-mesure avec les modules adaptés

**Résultat :**
Sophie a identifié les besoins en 10 minutes au lieu de faire 3 réunions. Elle peut créer un programme vraiment adapté.

---

### Scénario 3 : Quiz de fin de formation

**Situation :**
Thomas veut créer un quiz de validation des acquis pour sa formation PowerPoint.

**Étapes :**
1. Il crée un QCM de 20 questions sur Google Forms
2. Il copie l'URL
3. Dans Mentore Manager, il crée un nouveau questionnaire :
   - Titre : "Quiz PowerPoint - Validation"
   - Type : "Quiz / Exercice"
   - URL : son lien Google Forms
4. Il le lie à son module "PowerPoint Niveau 1"
5. Il enregistre

**Résultat :**
Maintenant, toutes ses formations PowerPoint ont automatiquement le lien vers le quiz final. Thomas peut le partager en fin de session pour valider les acquis.

## ✅ Conformité Qualiopi

Cette fonctionnalité vous aide à respecter les indicateurs Qualiopi suivants :

**Indicateur 3 : L'adaptation aux publics bénéficiaires**
- Les tests de niveau permettent d'adapter la formation au niveau initial
- Les audits des besoins permettent de personnaliser le contenu
- Preuve que vous évaluez AVANT de former

**Indicateur 6 : Les moyens pédagogiques et techniques**
- Les modalités d'évaluation sont documentées
- Les quiz et exercices font partie des moyens pédagogiques
- Preuve que vous évaluez PENDANT et APRÈS la formation

**Documentation générée automatiquement :**
- Liste des questionnaires utilisés par formation
- Type d'évaluation (positionnement, validation, besoins)
- Lien vers les questionnaires (preuve d'existence)
- Traçabilité : quel questionnaire pour quel programme/module

**Important pour Qualiopi :**
Mentore Manager stocke les liens, mais les RÉSULTATS des questionnaires sont dans Google Forms/Tally. Pensez à exporter et archiver les réponses pour vos audits.

## 🆚 Mentore Manager vs La concurrence

| Fonctionnalité | Digiforma | Qualiobee | Dendreo | Mentore Manager |
|----------------|-----------|-----------|---------|-----------------|
| **Créer un questionnaire** | Outil propriétaire complexe | Interface datée, 15 champs | Pas de lien externe, tout en interne | **Créez où vous voulez, Mentore stocke le lien** |
| **Google Forms / Tally** | Non supporté | Non supporté | Non supporté | **Oui, n'importe quel outil en ligne** |
| **Lien avec programmes** | Pas de lien automatique | Possible mais manuel | Compliqué | **1 clic, automatique partout** |
| **Catégorisation** | Tags désorganisés | Pas de types clairs | Trop de catégories | **3 types clairs : Niveau, Quiz, Audit** |
| **Ouverture du lien** | Copier-coller manuel | Export PDF puis recherche | Interface lourde | **1 clic, le lien s'ouvre** |

**Pourquoi Mentore Manager est différent :**
- ⚡ **Plus rapide** : Créer un questionnaire = 1 minute (contre 15 minutes pour créer le quiz dans les outils internes)
- 😊 **Plus fun** : Utilisez Google Forms ou Tally que vous connaissez déjà, pas besoin d'apprendre un nouvel outil
- 🎯 **Plus simple** : Mentore Manager fait juste le lien, vous gardez la main sur vos quiz

## ❓ Questions fréquentes

**Q: Est-ce que Mentore Manager crée les questionnaires pour moi ?**  
R: Non. Mentore Manager stocke juste le LIEN vers votre quiz. Vous créez le contenu sur Google Forms, Tally, Typeform... l'outil que vous préférez.

**Q: Où sont stockés les résultats des questionnaires ?**  
R: Dans l'outil où vous avez créé le quiz (Google Forms, Tally...). Mentore Manager ne stocke pas les réponses.

**Q: Puis-je lier un questionnaire à plusieurs programmes ?**  
R: Oui ! Un test de niveau Excel peut être lié à "Pack Office", "Formation Excel", "Bureautique Avancée", etc.

**Q: C'est quoi la différence entre les 3 types ?**  
R: **Test de niveau** = AVANT la formation (positionnement). **Quiz/Exercice** = PENDANT ou APRÈS (validation). **Audit des besoins** = AVANT (analyse besoins client).

**Q: Est-ce que je peux utiliser un questionnaire Word ou PDF ?**  
R: Techniquement oui (uploadez-le comme Support), mais c'est moins pratique. Google Forms permet de collecter les réponses automatiquement.

**Q: Le lien vers mon Google Forms a expiré, que faire ?**  
R: Modifiez le questionnaire et mettez le nouveau lien. Mentore Manager ne vérifie pas si le lien fonctionne.

**Q: Puis-je voir combien d'apprenants ont rempli le questionnaire ?**  
R: Non, cette information est dans Google Forms/Tally, pas dans Mentore Manager. Mentore Manager gère juste le lien.
