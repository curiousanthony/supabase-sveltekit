# Programmes

> **Pour qui ?** Directeurs, Commerciaux, Secrétaires
> 
> **En bref :** Assemblez vos modules pour créer des programmes complets avec prix, durée et pré-requis.
> 
> **Indicateurs Qualiopi :** Indicateur 6 (objectifs pédagogiques), Indicateur 11 (moyens pédagogiques)

---

## 🎯 Pourquoi cette fonctionnalité ?

Vous passez 30 minutes à recalculer la durée totale ? Vous oubliez des modules dans vos programmes ? Vous ressaisissez les mêmes informations pour chaque client ?

**Le problème avec les autres outils :**
- **Digiforma** : Pas de notion de programme, vous recréez tout pour chaque formation
- **Qualiobee** : Interface tellement complexe qu'il faut un tuto de 20 minutes pour comprendre
- **Dendreo** : Vous devez ressaisir manuellement tous les objectifs pédagogiques

**La solution Mentore Manager :**

Un programme, c'est comme une playlist. Vous sélectionnez vos modules (vos chansons), Mentore Manager calcule automatiquement la durée totale, assemble tous les objectifs pédagogiques, et génère votre programme complet. En 2 minutes, votre offre de formation est prête.

**Bénéfices clés :**
- ✅ **Assemblage ultra-rapide** : Glissez vos modules, c'est prêt en 2 minutes
- ✅ **Calcul automatique** : Durée totale calculée automatiquement, zéro erreur
- ✅ **Objectifs regroupés** : Tous les objectifs de vos modules assemblés automatiquement
- ✅ **Prix et modalités** : Ajoutez le prix public, la modalité (présentiel, distanciel...), les pré-requis
- ✅ **Statut de publication** : Brouillon, Publié, Archivé - gérez votre catalogue proprement

## 📋 Fonctionnalités principales

### Créer un programme

**C'est quoi ?**
C'est assembler plusieurs modules pour créer une offre de formation complète avec un prix et des informations commerciales.

**Pourquoi vous en avez besoin ?**
Parce que vos clients n'achètent pas "Excel Niveau 1" mais "Pack Office Complet" ou "Formation Bureautique Intensive". Le programme, c'est votre produit commercial.

**Comment l'utiliser ?**
1. Cliquez sur "Bibliothèque" → Onglet "Programmes"
2. Cliquez sur "Nouveau programme"
3. Remplissez les informations de base :
   - **Titre** : Le nom commercial (ex: "Pack Office Complet")
   - **Description** : Ce que contient le programme
   - **Modalité** : Cliquez sur la carte (Présentiel, Distanciel, Hybride, E-Learning)
   - **Statut** : Brouillon (par défaut) - vous publierez plus tard
   - **Durée** : Utilisez le stepper ou les boutons rapides (7h, 14h, 21h, 35h, 70h)
   - **Prix** : Le prix public en euros
4. Ajoutez vos modules :
   - Dans la liste "Modules disponibles", cliquez sur les modules que vous voulez
   - Ils s'ajoutent dans l'ordre
   - La durée des modules s'affiche à droite
5. Ajoutez les pré-requis (optionnel) :
   - Cochez ce qui est nécessaire : Ordinateur, Vidéoprojecteur, Connexion internet...
6. Cliquez sur "Créer le programme"

**Astuces :**
- 💡 **Ordre des modules** : Vous pouvez réorganiser avec les poignées de glissement (si besoin)
- 💡 **Durée flexible** : La durée totale peut être différente de la somme des modules (pauses, évaluations...)
- 💡 **Statut Brouillon** : Créez tranquillement, publiez quand c'est prêt

---

### Modifier un programme

**C'est quoi ?**
C'est changer le contenu, le prix, les modules d'un programme existant.

**Pourquoi vous en avez besoin ?**
Vos tarifs changent, vous voulez ajouter un module, retirer un module obsolète...

**Comment l'utiliser ?**
1. Dans la liste des programmes, cliquez sur le nom du programme
2. Modifiez ce que vous voulez
3. Cliquez sur "Enregistrer"

**Astuces :**
- 💡 **Ajoutez/retirez des modules** : Cliquez pour ajouter, cliquez sur X pour retirer
- 💡 **Changez les prix facilement** : Parfait pour les promotions ou ajustements
- 💡 **Statut "En cours"** : Utilisez-le pour les programmes en construction

---

### Dupliquer un programme

**C'est quoi ?**
C'est créer une copie d'un programme pour créer une variante.

**Pourquoi vous en avez besoin ?**
Vous avez "Pack Office Standard" et vous voulez créer "Pack Office Premium" avec des modules supplémentaires. Ou une version entreprise vs une version individuelle.

**Comment l'utiliser ?**
1. Dans la liste, cliquez sur les 3 points du programme
2. Cliquez sur "Dupliquer"
3. Un nouveau programme "Copie de [nom]" est créé avec statut "Brouillon"
4. Modifiez le nom, ajoutez/retirez des modules, changez le prix
5. Publiez

**Astuces :**
- 💡 **Variantes par public** : "Excel pour Débutants", "Excel pour Comptables"
- 💡 **Variantes par durée** : "Formation Intensive (35h)", "Formation Découverte (7h)"
- 💡 **Variantes par prix** : Standard, Premium, VIP avec supports et coaching

---

### Publier / Archiver un programme

**C'est quoi ?**
C'est gérer le cycle de vie de vos programmes : Brouillon → Publié → Archivé.

**Pourquoi vous en avez besoin ?**
Pour garder un catalogue propre. Les clients ne voient que les programmes "Publiés". Les anciens programmes sont "Archivés" mais conservés.

**Comment l'utiliser ?**
1. Ouvrez le programme
2. Changez le statut avec les boutons :
   - **Brouillon** : En construction, pas visible commercialement
   - **En cours** : En finalisation
   - **Publié** : Disponible pour vos clients et formations
   - **Archivé** : Ancien programme, conservé pour historique
3. Enregistrez

**Astuces :**
- 💡 **N'effacez jamais** : Archivez plutôt que supprimer, c'est mieux pour l'historique Qualiopi
- 💡 **Publiez quand c'est fini** : Prenez le temps de valider prix, modules, description
- 💡 **"En cours" temporaire** : Utilisez-le pendant les révisions

## 👤 Cas d'usage

### Scénario 1 : Créer un Pack Office Complet

**Situation :**
Sophie est commerciale. Un client lui demande une formation complète sur Office. Elle a déjà créé ses modules Word, Excel, PowerPoint.

**Étapes :**
1. Elle ouvre Bibliothèque → Programmes
2. Elle clique sur "Nouveau programme"
3. Elle remplit :
   - Titre : "Pack Office Complet"
   - Description : "Maîtrisez Word, Excel et PowerPoint de A à Z"
   - Modalité : Présentiel (elle clique sur la carte)
   - Prix : 1500€
4. Elle ajoute ses 3 modules :
   - Word Niveau 1
   - Excel Niveau 1
   - PowerPoint Niveau 1
5. Le système affiche "Durée modules : 21h"
6. Elle fixe la durée totale à 21h
7. Elle coche les pré-requis : Ordinateur, Vidéoprojecteur
8. Elle laisse le statut en "Brouillon" pour valider avec sa direction
9. Elle clique sur "Créer le programme"

**Résultat :**
En 2 minutes, Sophie a un programme complet avec tous les objectifs pédagogiques des 3 modules, la durée, le prix. Elle peut maintenant créer un devis ou une formation.

---

### Scénario 2 : Créer une variante Premium

**Situation :**
Marc veut créer une version "Pack Office Premium" avec des modules avancés en plus du Pack standard.

**Étapes :**
1. Il duplique "Pack Office Complet"
2. Le système crée "Copie de Pack Office Complet"
3. Il renomme : "Pack Office Premium"
4. Il ajoute 3 modules avancés :
   - Excel Tableaux croisés
   - PowerPoint Animation avancée
   - Outlook Gestion avancée
5. La durée passe à 35h
6. Il change le prix : 2200€
7. Il change le statut : "Publié"
8. Il enregistre

**Résultat :**
Marc a maintenant 2 offres : Standard (21h, 1500€) et Premium (35h, 2200€). Il peut proposer les deux à ses clients.

---

### Scénario 3 : Créer une formation sur-mesure

**Situation :**
Une entreprise veut une formation personnalisée : Excel + Power BI + Tableaux de bord.

**Étapes :**
1. Julie crée d'abord les modules manquants :
   - Power BI Initiation
   - Tableaux de bord efficaces
2. Elle crée un nouveau programme : "Business Intelligence & Reporting"
3. Elle ajoute ses 3 modules :
   - Excel Avancé
   - Power BI Initiation
   - Tableaux de bord efficaces
4. Durée totale : 21h
5. Prix sur-mesure : 1800€
6. Modalité : Hybride (2 jours présentiel, 1 jour distanciel)
7. Statut : "Publié"

**Résultat :**
En 10 minutes (dont 8 pour créer les nouveaux modules), Julie a une offre complète personnalisée, prête à être transformée en formation.

## 🆚 Mentore Manager vs La concurrence

| Fonctionnalité | Digiforma | Qualiobee | Dendreo | Mentore Manager |
|----------------|-----------|-----------|---------|-----------------|
| **Créer un programme** | Pas de programme, tout dans chaque formation | Configuration complexe 20+ clics | Interface confuse, besoin d'aide | **Glissez vos modules, 2 minutes** |
| **Calcul de durée** | Manuel, risque d'erreur | Calculatrice externe nécessaire | Semi-automatique mais bugs | **100% automatique, zéro erreur** |
| **Objectifs regroupés** | Copier-coller manuel de chaque module | Pas de regroupement automatique | Affichage mais pas d'export | **Export automatique de tous les objectifs** |
| **Variantes (Standard/Premium)** | Tout recréer à la main | Duplication partielle | Export/import fastidieux | **1 clic, dupliquer, modifier** |
| **Tarification** | Pas de gestion de prix | Prix par session, pas par programme | Prix caché dans config | **Prix public visible, modifiable facilement** |

**Pourquoi Mentore Manager est différent :**
- ⚡ **Plus rapide** : Créer un programme = 2 minutes vs 30 minutes ailleurs
- 😊 **Plus fun** : Glissez-déposez vos modules comme dans Spotify, c'est visuel et agréable
- 🎯 **Plus simple** : Le système calcule, assemble, génère tout. Vous, vous choisissez juste les modules

## ❓ Questions fréquentes

**Q: Quelle est la différence entre durée des modules et durée totale du programme ?**  
R: La durée des modules = somme des durées de chaque module. La durée totale = durée réelle avec pauses, évaluations, etc. Généralement, la durée totale est un peu supérieure.

**Q: Puis-je créer un programme sans modules ?**  
R: Oui techniquement, mais ce n'est pas l'idée. Un programme vide n'a pas d'objectifs pédagogiques ni de contenu. Créez au moins 1 module.

**Q: Combien de modules puis-je mettre dans un programme ?**  
R: Illimité ! Vous pouvez créer un programme avec 1 module ou 50 modules.

**Q: Si je modifie un module, le programme est automatiquement à jour ?**  
R: Oui ! C'est tout l'intérêt. Vous modifiez 1 module, tous les programmes avec ce module sont à jour.

**Q: C'est quoi la différence entre les statuts Brouillon, En cours, Publié ?**  
R: **Brouillon** = en construction. **En cours** = en finalisation. **Publié** = disponible pour créer des formations. **Archivé** = ancien programme conservé pour historique.

**Q: Puis-je exporter mes programmes ?**  
R: Pas encore directement, mais vous pouvez créer une formation à partir d'un programme et exporter le programme complet depuis la formation.

**Q: Les pré-requis sont obligatoires ?**  
R: Non, c'est optionnel. Mais c'est utile pour informer vos clients et c'est bon pour Qualiopi (indicateur 4).
