# Modules

> **Pour qui ?** Directeurs, Commerciaux, Secrétaires
> 
> **En bref :** Créez vos briques pédagogiques réutilisables avec objectifs, contenu et évaluation.
> 
> **Indicateurs Qualiopi :** Indicateur 6 (objectifs pédagogiques et moyens pédagogiques)

---

## 🎯 Pourquoi cette fonctionnalité ?

Vous recréez les mêmes contenus pour chaque formation ? Vous oubliez les objectifs pédagogiques ? Vous ne savez plus quelle durée mettre ?

**Le problème avec les autres outils :**
- **Digiforma** : Pas de modules réutilisables, vous ressaisissez tout pour chaque formation
- **Qualiobee** : Les modules existent mais sont trop techniques à créer
- **Dendreo** : Interface confuse avec 50 champs dont vous n'avez pas besoin

**La solution Mentore Manager :**

Un module, c'est une brique pédagogique. Pensez aux Lego : vous créez vos briques une fois, puis vous les assemblez pour construire vos programmes. Un module "Excel Niveau 1" peut servir dans "Pack Office", "Remise à niveau", "Formation Bureautique", etc.

**Bénéfices clés :**
- ✅ **Créez une fois, réutilisez partout** : Un module dans 10 programmes différents
- ✅ **Objectifs pédagogiques automatiques** : Qualiopi est content, vous aussi
- ✅ **Durée calculée automatiquement** : Fini les erreurs de calcul
- ✅ **Mise à jour centralisée** : Modifiez 1 module, toutes vos formations sont à jour

## 📋 Fonctionnalités principales

### Créer un module

**C'est quoi ?**
C'est remplir un formulaire simple avec le nom, le contenu, les objectifs et la durée de votre module.

**Pourquoi vous en avez besoin ?**
Pour arrêter de ressaisir les mêmes informations dans chaque formation. Créez une fois, réutilisez à l'infini.

**Comment l'utiliser ?**
1. Cliquez sur "Bibliothèque" dans le menu
2. Onglet "Modules"
3. Cliquez sur "Nouveau module"
4. Remplissez les champs :
   - **Titre** : Le nom de votre module (ex: "Excel Niveau 1")
   - **Contenu** : Ce que vous allez enseigner (ex: "Introduction aux fondamentaux, formules, tableaux...")
   - **Objectifs pédagogiques** : Ce que l'apprenant va savoir faire (ex: "Créer un tableau, utiliser les formules...")
   - **Modalité d'évaluation** : Comment vous évaluez (QCM, pratique, projet)
   - **Durée** : Combien d'heures (utilisez les boutons rapides : 3.5h, 7h, 14h...)
5. Cliquez sur "Créer le module"

**Astuces :**
- 💡 **Utilisez les placeholders** : Le formulaire vous donne des exemples conformes Qualiopi, copiez-les !
- 💡 **Pensez réutilisable** : Un module = 1 sujet. Ne mettez pas "Formation complète Excel" mais "Excel Niveau 1", "Excel Niveau 2"
- 💡 **Durée réaliste** : Mettez la durée réelle d'enseignement, pas la durée totale avec les pauses

---

### Modifier un module

**C'est quoi ?**
C'est mettre à jour un module existant. Les changements se répercutent partout où le module est utilisé.

**Pourquoi vous en avez besoin ?**
Votre contenu évolue (nouvelles versions d'Excel, nouvelles méthodes...). Plutôt que de modifier 10 formations, vous modifiez 1 module.

**Comment l'utiliser ?**
1. Dans la liste des modules, cliquez sur le nom du module
2. Modifiez ce que vous voulez
3. Cliquez sur "Enregistrer"

**Astuces :**
- 💡 **Attention aux anciennes formations** : La modification s'applique partout, même aux formations passées
- 💡 **Dupliquez si besoin** : Si vous voulez une nouvelle version sans changer l'ancienne, dupliquez d'abord
- 💡 **Historique automatique** : Mentore Manager garde l'historique des modifications (bon pour Qualiopi)

---

### Dupliquer un module

**C'est quoi ?**
C'est créer une copie d'un module existant pour le modifier sans toucher à l'original.

**Pourquoi vous en avez besoin ?**
Vous voulez créer "Excel 2024" sans modifier "Excel 2023" qui est encore utilisé dans des formations en cours.

**Comment l'utiliser ?**
1. Dans la liste des modules, cliquez sur les 3 points à droite
2. Cliquez sur "Dupliquer"
3. Un nouveau module "Copie de [nom]" est créé
4. Modifiez-le comme vous voulez

**Astuces :**
- 💡 **Renommez tout de suite** : "Copie de Excel Niveau 1" → "Excel Niveau 1 - 2024"
- 💡 **Utile pour les variantes** : Excel pour débutants, Excel pour comptables, Excel pour commerciaux...

---

### Supprimer un module

**C'est quoi ?**
C'est effacer définitivement un module que vous n'utilisez plus.

**Pourquoi vous en avez besoin ?**
Pour garder votre bibliothèque propre et ne pas vous perdre dans des centaines de modules obsolètes.

**Comment l'utiliser ?**
1. Ouvrez le module
2. En bas de page, section "Supprimer ce module"
3. Cliquez sur "Supprimer"
4. Confirmez

**Astuces :**
- 💡 **Attention !** : C'est irréversible. Le module disparaît de tous vos programmes
- 💡 **Préférez archiver** : Si le module est encore utilisé, mieux vaut le dupliquer et modifier la nouvelle version
- 💡 **Vérifiez l'usage** : Avant de supprimer, regardez dans quels programmes le module est utilisé

## 👤 Cas d'usage

### Scénario 1 : Créer un module Word Débutant

**Situation :**
Marie est formatrice bureautique. Elle donne régulièrement des formations Word. Elle veut créer un module réutilisable.

**Étapes :**
1. Elle ouvre la Bibliothèque, onglet Modules
2. Elle clique sur "Nouveau module"
3. Elle remplit :
   - Titre : "Word Débutant"
   - Contenu : "Présentation de l'interface, création de documents, mise en forme basique, insertion d'images et tableaux"
   - Objectifs : "À l'issue de ce module, le stagiaire sera capable de créer un document, le mettre en forme, insérer des éléments"
   - Modalité d'évaluation : "Pratique" (elle clique sur la carte)
   - Durée : 7h (elle clique sur le bouton "7h")
4. Elle clique sur "Créer le module"

**Résultat :**
En 3 minutes, Marie a créé un module complet. Elle peut maintenant l'utiliser dans tous ses programmes de formation Word.

---

### Scénario 2 : Mettre à jour un module Excel

**Situation :**
Sophie utilise un module "Excel Tableaux croisés" depuis 1 an. Microsoft vient de sortir de nouvelles fonctionnalités. Elle veut mettre à jour le module.

**Étapes :**
1. Elle ouvre son module "Excel Tableaux croisés"
2. Elle modifie les objectifs pédagogiques pour ajouter les nouvelles fonctions
3. Elle ajoute dans le contenu : "Nouveautés Excel 2024 : tableaux croisés dynamiques"
4. Elle enregistre

**Résultat :**
Toutes les formations futures avec ce module auront automatiquement le contenu à jour. Sophie n'a pas besoin de modifier 15 programmes un par un.

---

### Scénario 3 : Créer une déclinaison d'un module

**Situation :**
Thomas a un module "PowerPoint Niveau 1" pour tout le monde. Il veut créer une version spéciale pour les commerciaux avec des techniques de pitch.

**Étapes :**
1. Il duplique "PowerPoint Niveau 1"
2. Le système crée "Copie de PowerPoint Niveau 1"
3. Il renomme : "PowerPoint pour Commerciaux"
4. Il modifie les objectifs pour ajouter : "Créer un pitch commercial percutant"
5. Il modifie le contenu pour ajouter des exemples de présentation commerciale
6. Il enregistre

**Résultat :**
Thomas a maintenant 2 modules : un générique et un spécialisé pour commerciaux. Il peut utiliser les deux dans différents programmes.

## 🆚 Mentore Manager vs La concurrence

| Fonctionnalité | Digiforma | Qualiobee | Dendreo | Mentore Manager |
|----------------|-----------|-----------|---------|-----------------|
| **Créer un module** | Pas de module réutilisable | Interface technique, 15 champs | Menu confus, 8 étapes | **1 formulaire, 5 champs, 2 minutes** |
| **Objectifs pédagogiques** | Texte libre, vous vous débrouillez | Champ texte basique | Template rigide | **Placeholder Qualiopi inclus, copiez-collez !** |
| **Durée** | Saisie manuelle avec risque d'erreur | Calculatrice externe nécessaire | Format bizarre (1h30 = 1.5 ou 1.30 ?) | **Stepper intuitif + boutons rapides** |
| **Modalité d'évaluation** | Liste déroulante cachée | Pas clair | Jargon technique | **4 cartes visuelles : QCM, QCU, Pratique, Projet** |
| **Réutilisation** | Impossible, copier-coller manuel | Possible mais compliqué | Export/import fastidieux | **1 clic dans vos programmes** |

**Pourquoi Mentore Manager est différent :**
- ⚡ **Plus rapide** : 2 minutes pour créer un module vs 15 minutes ailleurs
- 😊 **Plus fun** : Interface visuelle avec cartes et boutons, pas de liste déroulante sans fin
- 🎯 **Plus simple** : Les placeholders vous guident pour être conforme Qualiopi directement

## ❓ Questions fréquentes

**Q: Combien de modules puis-je créer ?**  
R: Illimité ! Créez autant de modules que vous voulez. Pas de limite.

**Q: Est-ce que je dois remplir tous les champs ?**  
R: Non. Seul le titre est obligatoire. Mais pour Qualiopi, on recommande de remplir au minimum : titre, objectifs, durée.

**Q: Puis-je utiliser le même module dans plusieurs programmes ?**  
R: Oui, c'est fait pour ça ! Un module peut être dans 10, 20, 100 programmes différents.

**Q: Si je modifie un module, ça modifie mes anciennes formations ?**  
R: Oui. Si vous ne voulez pas modifier les anciennes, dupliquez d'abord le module, puis modifiez la copie.

**Q: C'est quoi la différence entre Contenu et Objectifs pédagogiques ?**  
R: **Contenu** = ce que vous enseignez (ex: "Les formules Excel"). **Objectifs** = ce que l'apprenant saura faire (ex: "Créer une formule SOMME"). Pour Qualiopi, les objectifs sont plus importants.

**Q: Pourquoi il y a des exemples dans les champs ?**  
R: Ce sont des placeholders conformes Qualiopi. Vous pouvez les copier et adapter. Ça vous fait gagner du temps et vous assure d'être conforme.

**Q: Est-ce que Mentore Manager vérifie que mes objectifs sont conformes Qualiopi ?**  
R: Pas encore automatiquement, mais les placeholders vous guident vers la bonne formulation. Un objectif Qualiopi commence par un verbe d'action ("créer", "utiliser", "évaluer"...).
