---
description: Push changes and create a Pull Request
---

1.  **Push Branch**:
    ```bash
    git push -u origin HEAD
    ```

2.  **Notify User**:
    -   Inform the user that the branch is pushed.
    -   Provide the link to create the PR (GitHub usually outputs this).
    -   Remind them to use "Squash and Merge" when merging.
